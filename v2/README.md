# CrowdWalrus Docs

Welcome to **CrowdWalrus** — a decentralized crowdfunding dApp built on **Sui**.
Campaign actions (create / update / contribute) are on-chain, while rich content is stored on **Walrus** for censorship-resistant publishing. Human-readable names are powered by **SuiNS**.

> No gatekeepers. No middlemen. Just you and your supporters.
