# Install a Sui wallet

To use CrowdWalrus, you need a wallet that can:
- connect to Sui dApps
- sign transactions
- hold SUI (gas) and other Sui tokens

## Recommended: Slush (formerly Sui Wallet)
Slush is the Mysten Labs wallet app for the Sui ecosystem.

- Browser extension
- Mobile apps
- Supports dApp connections and network switching

Get it here:
- https://slush.app/

- Use a hardware wallet for serious funds.
