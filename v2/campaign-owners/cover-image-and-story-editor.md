# Cover image + rich story editor

Your campaign story is your conversion engine.

## Cover image
- Use JPG/PNG
- Keep it under ~5MB
- Choose something that still looks good on mobile

## Rich story editor
The editor supports rich formatting and images.

**Where this is stored:** Walrus
- the editor state is stored as JSON
- images are stored as blobs

### Writing tips
- Start with a 1–2 sentence “why now?”
- Add a clear breakdown of where funds go
- Include proof links (repo, docs, prior work)
